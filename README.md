## How to Test
The backend is deployed live on Render. You can use Postman to test these endpoints:
* **Create User:** `POST` /api/users
* **Create Group:** `POST` /api/groups
* **Add Expense:** `POST` /api/expenses
* **Check Balance:** `GET` /api/groups/:id/balance
